This repository contains MATLAB code that was used for the paper 'Master Equation Reduction using State Projection and Singular
Perturbation Arguments' by Alec Elias Sigurdarson and Michail Stamatakis. The files SingularPerturbationNumericalExample.m and 
SingularPerturbationLattice2D.m contain the main code that should be run to simulate the systems considered in the paper. The 
files plots.m and uneqplot.m are additional scripts used for plotting. The other files contain functions and parameters 
necessary for running the main code.
